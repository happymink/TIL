# 📃 My Commit Template

### ✉ Commit Message Style
```
  [TYPE] message
```

<br>

### 🔎 Commit Type
|Type|Usage|
|:----:|:--------|
|ADD|새로운 기능 추가|
|DELETE|삭제|
|FIX|에러 수정, 잘못된 것 수정|
|MODIFY|기능 수정|
|NOTE|주석|
|REFACTOR|리팩토링|
|STYLE|코드 스타일 수정|
|TEST|테스트 코드|
|UPDATE|문서, 레이아웃, 기타 수정|
